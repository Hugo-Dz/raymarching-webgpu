## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Npm](https://docs.npmjs.com/getting-started)

### Run it locally

- Clone the project.

```bash
    git clone https://github.com/Hugo-Dz/raymarching-webgpu.git
    cd raymarching-webgpu
```

- Install the dependencies.

```bash
    npm install
```

- Start the dev server

```bash
    npm run dev
```
